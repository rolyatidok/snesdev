YY-CHR.NET用のプラグイン作成用のプロジェクト/ソースです。
開発環境は Visual Studio 2008/2010 Expressです。

・C#版はプロジェクトをzipファイル化しています。
　zipファイルの展開後、プロジェクトを開き、
　YY-CHR.netに付属の"CharactorLib.dll"の参照を追加すると、コンパイルできます。

・VB.NET版はソースファイルです。
　zipファイルを展開するとvbソースが出てくるので、
　新しいプロジェクトにクラスライブラリを指定して作成し、ソースをプロジェクトに追加してください。
　YY-CHR.netに付属の"CharactorLib.dll"の参照を追加すると、コンパイルできます。

--------------------------------------------------------------------------------

●サンプルソースは「2BPP NES」と同等の動きをする画像形式のプラグインです。

●プロジェクトの参照設定にYY-CHR.netに付属の"CharactorLib.dll"を含めてください。
　動作させるYY-CHRに付属しているdllである必要があります。

●作成したプラグインはYY-CHRのexeがあるディレクトリの「plugins」ディレクトリに入れます。
 「plugins」ディレクトリが無い場合は作成してください。
　YY-CHRを起動すると、画像形式のコンボボックスの一番下にプラグインに含まれる形式が追加されます。
　FormatBaseを継承したクラスが複数あれば、全て追加されます。

●現バージョンのYY-CHRでは画像形式は変換後の画像サイズが128x128になるようにしてください。

その他のプラグインの状況

■対応済み
FormatBase：画像形式

■実装中
DataFileBase：外部ファイル(PalやDatやADFのような外部ファイルを扱うクラス)
EditFunctionBase：ペンやライン、矩形円などの描画処理

■その他
リフレクションを使用することで、プラグインからGUIを無理やり拡張できるかもしれません。
その場合、メニューの [Help > Property Editor] でYY-CHRのコントロールや変数を表示・変更することができます。
プロパティエディタは、画像形式のプラグインのデバッグでも使えるかもしれません。
使用中の画像形式のFormatBaseクラスは mFormat で参照できます。
